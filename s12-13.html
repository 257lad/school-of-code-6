Session 12
DOM Manipulation

Le DOM (Document Object Model) est la représentation en mémoire de ta page HTML. Quand le navigateur charge une page, il transforme chaque balise HTML en un objet JavaScript que tu peux lire et modifier.

Grâce au DOM, tu peux : changer du texte, modifier des styles, ajouter ou supprimer des éléments — tout ça sans recharger la page.

Sélectionner un élément — pour le manipuler, il faut d'abord le trouver.
Sélectionner des éléments 
// Par ID — retourne un seul élément
    const titre = document.getElementById('titre');
    
    // Par classe — retourne une liste
    const cartes = document.getElementsByClassName('carte');
    
    // Sélecteur CSS — le plus moderne et polyvalent
    const btn   = document.querySelector('#monBtn');
    const tous  = document.querySelectorAll('.item');
Modifier le contenu et le style — une fois sélectionné, on peut tout changer.
Modifier contenu & style 
const el = document.querySelector('#boite');
    
    // Changer le texte
    el.textContent = "Nouveau texte";
    
    // Changer le HTML interne
    el.innerHTML = "<strong>Texte en gras</strong>";
    
    // Modifier le style CSS
    el.style.color = "blue";
    el.style.fontSize = "20px";
    
    // Ajouter / retirer une classe CSS
    el.classList.add('actif');
    el.classList.remove('actif');
    el.classList.toggle('actif');
Créer et supprimer des éléments — on peut construire le HTML dynamiquement.
Créer & insérer des éléments 
// Créer un nouvel élément
    const li = document.createElement('li');
    li.textContent = "Nouvelle tâche";
    
    // L'ajouter dans le DOM
    const liste = document.querySelector('#maListe');
    liste.appendChild(li);
    
    // Supprimer un élément
    li.remove();

Session 12
Events — Les Événements

Un événement est une action qui se produit dans le navigateur : un clic, une frappe au clavier, le survol d'un élément... On y réagit avec addEventListener.

addEventListener — structure de base 
element.addEventListener('événement', function(e) {
      // code à exécuter quand l'événement se produit
    });
Événements les plus courants
click — clic sur un élément
submit — envoi d'un formulaire
input — frappe dans un champ
keydown — touche pressée
mouseover — survol souris
DOMContentLoaded — page chargée
Exemples concrets 
const btn = document.querySelector('#monBouton');
    
    // Événement clic
    btn.addEventListener('click', (e) => {
      console.log('Bouton cliqué !');
    });
    
    // Récupérer la valeur d'un champ en temps réel
    const input = document.querySelector('#monInput');
    input.addEventListener('input', (e) => {
      console.log(e.target.value);
    });
    
    // Empêcher le comportement par défaut (ex: rechargement du form)
    const form = document.querySelector('form');
    form.addEventListener('submit', (e) => {
      e.preventDefault();
      console.log('Formulaire soumis sans rechargement !');
    });

Session 13
LocalStorage

Le LocalStorage est un espace de stockage dans le navigateur. Il permet de sauvegarder des données qui persistent même après avoir fermé et rouvert l'onglet — sans base de données ni serveur.

Limite : les données sont sous forme de texte uniquement. Pour stocker des objets ou tableaux, il faut les convertir avec JSON.stringify() et les relire avec JSON.parse().

localStorage.setItem(clé, valeur)Sauvegarder
Exemple 
// Valeur simple
    localStorage.setItem('prenom', 'Jean');
    
    // Tableau ou objet → JSON.stringify()
    const taches = ['Coder', 'Tester'];
    localStorage.setItem('taches', JSON.stringify(taches));
localStorage.getItem(clé)Lire
Exemple 
const prenom = localStorage.getItem('prenom');
    console.log(prenom);  // → "Jean"
    
    // Tableau → JSON.parse() pour retrouver le vrai tableau
    const sauvegarde = localStorage.getItem('taches');
    const taches = JSON.parse(sauvegarde);
    console.log(taches);  // → ["Coder", "Tester"]
localStorage.removeItem(clé) & .clear()Supprimer
Exemple 
// Supprimer une clé
    localStorage.removeItem('prenom');
    
    // Tout effacer
    localStorage.clear();
