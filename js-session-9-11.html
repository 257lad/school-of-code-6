Partie 1
C'est quoi une variable ?

Imagine que la mémoire de ton ordinateur est une grande étagère avec des boîtes. Une variable, c'est l'une de ces boîtes. Tu lui donnes :

→ un nom (pour la retrouver),
→ une valeur (ce qu'elle contient),
→ et un type (const ou let) si sa contenu peut changer ou non.

Boîte verrouillée — const

const pays
"Burundi"
String
❌ Ne peut jamais changer

Boîte ouverte — let

let score
10
Number
✅ Peut être modifiée
Déclarer et utiliser des variables 
// ─── const : valeur fixe, ne change JAMAIS
const pays        = "Burundi";
const PI          = 3.14;
// pays = "Rwanda"; ← ERREUR TypeError !

// ─── let : valeur modifiable
let score         = 10;
score             = 25;   //  OK, score vaut maintenant 25

// ─── typeof : connaître le type d'une variable
console.log(typeof pays);   // "string"
console.log(typeof score);  // "number"
console.log(typeof true);   // "boolean"
Les types de données
Type Ce que ça représente Exemples
String Du texte, entre guillemets simples, doubles ou backticks "Jean" 'Salut' `Bonjour`
Number N'importe quel nombre, entier ou décimal 42 3.14 -7
Boolean Uniquement vrai ou faux — pour les décisions true false
Undefined Variable déclarée mais sans valeur assignée let x; → undefined
Null Absence intentionnelle de valeur (on le choisit) let vide = null;
Array Une liste ordonnée d'éléments ["Mangue", "Banane"]
Object Un ensemble de paires clé : valeur { nom: "Jean", age: 25 }
Partie 2
Les Boucles

Une boucle répète un bloc de code plusieurs fois.

Anatomie d'une boucle for
for  (  let i = 0 ;  i <= 5 ;  i++  ) { ... }
① Variable d'induction
let i = 0 — le compteur. On le crée ici avec sa valeur de départ. Il évolue à chaque tour de boucle.
② Condition de terminaison
i <= 5 — la boucle tourne TANT QUE cette condition est vraie. Dès qu'elle est fausse → la boucle s'arrête.
③ Incrémentation / Décrémentation
i++ ajoute 1 après chaque tour. i-- enlève 1. Sans ça → boucle infinie !
Boucle for

À utiliser quand on connaît à l'avance le nombre de répétitions.

Incrémentation — i++ 
// Compte de 1 à 5
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
// → 1  2  3  4  5
Décrémentation — i-- 
// Compte à rebours de 5 à 1
for (let i = 5; i >= 1; i--) {
  console.log(i);
}
// → 5  4  3  2  1
Parcourir un tableau avec for 
const fruits = ["Mangue", "Ananas", "Banane"];

for (let i = 0; i < fruits.length; i++) {
  console.log("Fruit :", fruits[i]);
}
// → Fruit : Mangue
// → Fruit : Ananas
// → Fruit : Banane
Boucle while

À utiliser quand on ne connaît pas à l'avance le nombre de répétitions — on tourne tant qu'une condition est vraie.

Boucle while — exemple 
let compteur = 1;

while (compteur <= 3) {
  console.log("Tour numéro", compteur);
  compteur++;   // INDISPENSABLE pour éviter la boucle infinie
}
// → Tour numéro 1
// → Tour numéro 2
// → Tour numéro 3
⚠️

Danger : si tu oublies d'incrémenter le compteur dans un while, la condition ne devient jamais fausse → la boucle tourne à l'infini et bloque le navigateur !

Boucle do...while

La grande différence avec while : le bloc de code s'exécute au moins une fois, même si la condition est fausse dès le départ. On vérifie la condition après le premier tour.

while — vérifie AVANT

Si la condition est fausse dès le départ, le code ne s'exécute jamais.

do…while — vérifie APRÈS

Le code s'exécute toujours au moins une fois, puis vérifie la condition.

do...while — structure et exemple 
// Structure
do {
  // code exécuté au moins une fois
} while (/* condition */);

// ─── Exemple concret : demander un mot de passe
let tentative = 0;

do {
  console.log("Essai numéro", tentative + 1);
  tentative++;
} while (tentative < 3);

// → Essai numéro 1
// → Essai numéro 2
// → Essai numéro 3

// ─── Même avec condition fausse dès le départ :
let x = 10;
do {
  console.log("Exécuté quand même !");
} while (x < 5);   // condition fausse, mais...
// → "Exécuté quand même !" (une seule fois)
Partie 3
Les Opérateurs
Opérateurs Arithmétiques

Servent à faire des calculs avec des nombres.

+
Addition
Additionne deux nombres. Avec des strings : les colle ensemble (concaténation).
let a = 10 + 5; // → 15
let msg = "Salut " + "Jean"; // → "Salut Jean"
-
Soustraction
let reste = 20 - 8; // → 12
*
Multiplication
let produit = 6 * 7; // → 42
/
Division
let div = 15 / 3; // → 5
%
Modulo (reste)
Retourne le reste d'une division entière. Très utile pour savoir si un nombre est pair ou impair.
let r = 10 % 3; // → 1 (10 = 3×3 + 1)
let pair = 8 % 2; // → 0 (8 est pair car reste = 0)
**
Puissance
Élève un nombre à la puissance d'un autre. Ex : 2³ = 2 ** 3.
let p = 2 ** 3; // → 8 (2³)
let p2 = 5 ** 2; // → 25 (5²)
Opérateurs d'Affectation

Servent à attribuer ou mettre à jour la valeur d'une variable.

=
Assignation simple
Affecte une valeur à une variable.
let score = 10; // score vaut 10
+=
Addition et assignation
Ajoute une valeur à la variable ET la réassigne. Raccourci de x = x + n.
let score = 10;
score += 5; // score = 10 + 5 → 15
-=
Soustraction et assignation
Soustrait une valeur de la variable ET la réassigne.
let vie = 100;
vie -= 20; // vie = 100 - 20 → 80
*=
Multiplication et assignation
Multiplie la variable par une valeur ET la réassigne.
let prix = 50;
prix *= 2; // prix = 50 × 2 → 100
++ --
Incrémentation / Décrémentation
Ajoute ou enlève exactement 1. Raccourcis très utilisés dans les boucles.
let i = 5;
i++; // i = 5 + 1 → 6 (équivaut à i += 1)
i--; // i = 6 - 1 → 5 (équivaut à i -= 1)
Opérateurs de Comparaison

Comparent deux valeurs et retournent toujours true ou false. Utilisés dans les if et les boucles.

==
Égalité de valeur (souple)
Vérifie si les valeurs sont égales, mais ignore le type. Peut donner des résultats surprenants.
console.log(5 == "5"); // → true ⚠️ nombre == string !
===
Égalité stricte ✅ Recommandée
Vérifie valeur ET type en même temps. Toujours préférer ceci.
console.log(5 === 5); // → true
console.log(5 === "5"); // → false (types différents)
!= !==
Inégalité (souple / stricte)
!= ignore le type, !== vérifie aussi le type. Préférer !==.
console.log(5 !== 10); // → true (5 et 10 sont différents)
console.log(5 !== 5); // → false (identiques)
> <
Supérieur / Inférieur strict
Vérifie si une valeur est strictement plus grande ou plus petite.
console.log(10 > 5); // → true
console.log(3 < 8); // → true
console.log(5 > 5); // → false (égaux, pas strictement supérieur)
>= <=
Supérieur ou égal / Inférieur ou égal
Inclut le cas d'égalité dans la comparaison.
console.log(18 >= 18); // → true (18 est bien ≥ 18)
console.log(5 <= 3); // → false
Opérateurs Logiques

Combinent plusieurs conditions ensemble. Toujours utilisés dans les if.

&&
ET logique (AND)
Retourne true uniquement si toutes les conditions sont vraies en même temps.
let age = 20, carte = true;
if (age >= 18 && carte) {
  console.log("Accès autorisé"); // → s'exécute (les deux sont vrais)
}
||
OU logique (OR)
Retourne true si au moins une condition est vraie.
let admin = false, proprio = true;
if (admin || proprio) {
  console.log("Peut modifier"); // → s'exécute (proprio est vrai)
}
!
NON logique (NOT)
Inverse la valeur booléenne : transforme true en false, et false en true.
let connecte = false;
console.log(!connecte); // → true (inverse de false)
console.log(!true); // → false
Partie 4
Méthodes des Tableaux (Arrays)

Un tableau est une liste ordonnée. JavaScript offre de nombreuses méthodes intégrées pour le manipuler sans avoir à écrire des boucles à la main.

.push() & .pop() Array Ajouter / Supprimer à la fin

.push() ajoute un élément à la fin du tableau. .pop() supprime et retourne le dernier élément.

Exemple 
let fruits = ["Mangue", "Ananas"];

fruits.push("Banane");
console.log(fruits);  // → ["Mangue", "Ananas", "Banane"]

let dernier = fruits.pop();
console.log(dernier);  // → "Banane"
console.log(fruits);   // → ["Mangue", "Ananas"]
.unshift() & .shift() Array Ajouter / Supprimer au début

.unshift() ajoute au début. .shift() supprime et retourne le premier élément.

Exemple 
let liste = ["B", "C"];

liste.unshift("A");
console.log(liste);   // → ["A", "B", "C"]

liste.shift();
console.log(liste);   // → ["B", "C"]
.forEach() Array Parcourir chaque élément

Exécute une fonction pour chaque élément du tableau. Remplace avantageusement la boucle for classique.

Exemple 
const nombres = [1, 2, 3, 4];

nombres.forEach((n) => {
  console.log(n * 2);
});
// → 2   4   6   8
.map() Array Transformer chaque élément → nouveau tableau

Crée un nouveau tableau en appliquant une transformation à chaque élément. L'original n'est pas modifié.

Exemple 
const prix    = [100, 200, 300];
const doubles = prix.map((p) => p * 2);

console.log(doubles);  // → [200, 400, 600]
console.log(prix);     // → [100, 200, 300] (inchangé)
.filter() Array Garder uniquement certains éléments

Crée un nouveau tableau avec seulement les éléments qui satisfont une condition.

Exemple 
const ages   = [12, 20, 15, 30, 8];
const adultes = ages.filter((a) => a >= 18);

console.log(adultes);  // → [20, 30]
.find() .includes() .indexOf() Array Rechercher dans un tableau
Exemple 
const notes = [12, 15, 8, 18];

// .find() — retourne le 1er élément qui satisfait la condition
const bonne = notes.find((n) => n >= 15);
console.log(bonne);     // → 15

// .includes() — est-ce que la valeur existe ? (true/false)
console.log(notes.includes(8));   // → true
console.log(notes.includes(20));  // → false

// .indexOf() — à quel index se trouve la valeur ? (-1 si absent)
console.log(notes.indexOf(18));  // → 3
.reduce() Array Réduire à une seule valeur

Parcourt tout le tableau et accumule un résultat unique (somme, total, etc.).

Exemple — calculer le total 
const prix  = [50, 30, 20];
const total = prix.reduce((acc, p) => acc + p, 0);
//                          ↑ accumulateur  ↑ valeur de départ

console.log(total);  // → 100
.sort() & .reverse() Array Trier et inverser
Exemple 
const lettres = ["C", "A", "B"];
lettres.sort();
console.log(lettres);   // → ["A", "B", "C"]

lettres.reverse();
console.log(lettres);   // → ["C", "B", "A"]

// Pour trier des nombres correctement :
const n = [30, 10, 20];
n.sort((a, b) => a - b);
console.log(n);          // → [10, 20, 30]
Partie 5
Méthodes des Objets

Les objets stockent des données sous forme de clé : valeur. JavaScript propose plusieurs méthodes pour les explorer et les manipuler facilement.

Object.keys() .values() .entries() Object

Transforment un objet en tableau pour pouvoir le parcourir facilement.

Exemple 
const user = { nom: "Jean", age: 25, ville: "Bujumbura" };

// Object.keys() — tableau des clés
console.log(Object.keys(user));
// → ["nom", "age", "ville"]

// Object.values() — tableau des valeurs
console.log(Object.values(user));
// → ["Jean", 25, "Bujumbura"]

// Object.entries() — tableau de paires [clé, valeur]
console.log(Object.entries(user));
// → [["nom","Jean"], ["age",25], ["ville","Bujumbura"]]
Ajouter · Modifier · Supprimer une propriété Object
Exemple 
const user = { nom: "Jean", age: 25 };

// Ajouter une nouvelle propriété
user.ville = "Bujumbura";

// Modifier une propriété existante
user.age = 26;

// Supprimer une propriété
delete user.ville;

console.log(user);  // → { nom: "Jean", age: 26 }
Spread operator { ...objet } Object Copier & fusionner des objets

L'opérateur ... permet de copier ou fusionner des objets proprement, sans modifier l'original.

Exemple 
const base   = { nom: "Jean", age: 25 };
const extra  = { ville: "Bujumbura", job: "Dev" };

// Copier un objet
const copie  = { ...base };

// Fusionner deux objets
const complet = { ...base, ...extra };
console.log(complet);
// → { nom: "Jean", age: 25, ville: "Bujumbura", job: "Dev" }

// Modifier une propriété en copiant
const misAJour = { ...base, age: 30 };
console.log(misAJour);  // → { nom: "Jean", age: 30 }
Déstructuration const { clé } = objet Object Extraire des propriétés rapidement

Permet d'extraire des propriétés d'un objet directement dans des variables, sans répéter objet.propriete partout.

Exemple 
const user = { nom: "Jean", age: 25, ville: "Bujumbura" };

// Sans déstructuration (verbeux)
const nom1  = user.nom;
const age1  = user.age;

// Avec déstructuration (propre et rapide)
const { nom, age, ville } = user;

console.log(nom);    // → "Jean"
console.log(age);    // → 25
console.log(ville);  // → "Bujumbura"
.hasOwnProperty() & "clé" in objet Object Vérifier l'existence d'une propriété
Exemple 
const user = { nom: "Jean", age: 25 };

console.log(user.hasOwnProperty("nom"));    // → true
console.log(user.hasOwnProperty("email"));  // → false

console.log("age"   in user);   // → true
console.log("email" in user);   // → false
